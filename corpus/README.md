# Corpus — access & consent

The retrieval corpus is a **personal photo archive of 227 images** belonging to the author,
spanning ~25 years and centred on one recurring subject.

## Access for grading
- The 227 filenames (and their generated captions) are enumerated in
  [`../artifacts/captions.json`](../artifacts/captions.json).
- **Add before submission:** place a representative **20-image sample** in `corpus/sample/`
  (create the folder and drop the images in), or link a shared folder here, so reviewers can
  see the material without the full archive.

## Where to put the full corpus for the zip
When packaging the submission, add the full image set under `corpus/images/` (or `corpus/sample/`
for the 20-image subset) inside this folder, then compress the whole `seeing-machines/` directory.
The derived artifacts (captions, embeddings, comparison + evaluation results) are already committed;
the images themselves are the only thing this folder is waiting on.

## Rights & consent
- All images are the author's own photographs.
- Any other identifiable people appear with consent for coursework use.
- The archive is **not** redistributed publicly; only the derived artifacts
  (captions, embeddings, comparison and evaluation results) are committed to this repo.
